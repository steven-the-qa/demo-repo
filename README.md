### Prerequisites
- Node.js (v20 or higher)
- npm (comes with Node.js)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/socialqa/playwright-demo.git
```

2. Navigate to the project directory:
```bash
cd demo-repo
```

3. Install dependencies:
```bash
npm install
```

4. Run tests:
```bash
npx playwright test
```
